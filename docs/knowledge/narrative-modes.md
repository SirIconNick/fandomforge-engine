# Narrative Modes

Every multifandom edit tells a story, whether the editor thought about it or not. If you don't pick a narrative mode consciously, your edit will land on "clip compilation." Picking one deliberately is how edits go from compilations to little films.

## The four modes

### Convergent
**"Many roads, one destination."**

Each fandom starts in its own world, and the edit pulls them all toward one convergent point. The finale is a moment where all fandoms reach the same emotional or visual peak simultaneously.

**Structural signal:** acts 1 and 2 rotate through fandoms, act 3 intercuts all of them.

**Example theme:** "All heroes who knew they wouldn't survive." Act 1 — Tony introduces his arc. Act 2 — Obi-Wan introduces his. Act 3 — Erwin Smith introduces his. Act 4 — all three sacrifices intercut in quickfire on the drop.

**Strengths:**
- Huge emotional payoff when the convergence lands
- Viewer learns the rule ("these fandoms rhyme") and the payoff feels earned

**Risks:**
- If the convergence point is weak, the whole edit falls flat
- Requires the viewer to stay patient through act 2

### Parallel
**"The same story, told five times."**

Each fandom tells the same thematic beat in its own voice, sequentially. Then maybe another beat, again across all fandoms.

**Structural signal:** sections defined by theme beat, each section containing a fandom sweep.

**Example theme:** "The first time they lost someone." Section 1 — loss introductions (one per fandom). Section 2 — mourning (one per fandom). Section 3 — resolve (one per fandom). Etc.

**Strengths:**
- Theme reinforced through repetition
- Each fandom gets equal treatment
- Works well for thematic essays

**Risks:**
- Can feel monotonous if not paced
- Each "sweep" has to vary or the pattern tires

### Relay
**"You tell this part, I'll tell the next."**

Fandom A owns act 1. Fandom B owns act 2. Fandom C owns act 3.

**Structural signal:** acts defined by fandom ownership.

**Example theme:** "The hero's journey." Act 1 — Star Wars (the call). Act 2 — LOTR (the trial). Act 3 — Marvel (the return).

**Strengths:**
- Each fandom gets proper room
- Clean, easy to follow

**Risks:**
- Feels like three short edits stapled together unless visual language (color, transitions, audio) binds them
- The handoffs between acts need to be exceptionally strong match cuts

### Woven
**"They're all in the same scene now."**

Clips from all fandoms interleave throughout. No one fandom owns any section.

**Structural signal:** rapid cross-cutting throughout, no long stretches within one fandom.

**Example theme:** "Parkour through universes." Every 1-2 shots is a different fandom. The edit feels like one continuous action sequence assembled from 10 movies.

**Strengths:**
- Cinematic, virtuosic feel
- Theme reinforced by relentless juxtaposition

**Risks:**
- Requires tight matching — color, motion direction, framing
- Hardest mode to execute well
- Can become chaotic / visual soup without discipline

## Picking the right mode

| Situation | Best mode |
|---|---|
| Epic finale moment planned | Convergent |
| Thematic essay / philosophical point | Parallel |
| Each fandom has distinct tone | Relay |
| Action-forward, pure momentum | Woven |
| Short-form (under 60s) | Woven or tight Parallel |
| Long-form (3-5min) | Convergent or Relay |
| Newer editor | Relay (easiest) |
| Advanced editor | Woven (hardest, most rewarding) |

## Mode-specific treatment implications

### Convergent
- **Color:** source-proud at first, gradually unifying toward the convergence
- **Transitions:** hard cuts early, match cuts rising into the convergence
- **Audio:** let the song build naturally; the convergence lands on the drop

### Parallel
- **Color:** can keep each fandom slightly distinct — it helps viewer track who's who
- **Transitions:** consistent transition type per section to reinforce the pattern
- **Audio:** use SFX to reset between sections; each sweep is its own breath

### Relay
- **Color:** unified master grade across all acts, or intentional color shifts marking act changes
- **Transitions:** handoff moments between acts must be the strongest transitions (match cuts, dip to black)
- **Audio:** consistent song the whole way; SFX can differentiate acts

### Woven
- **Color:** unified grade, non-negotiable. Unmatched color will read as mistake here.
- **Transitions:** mostly hard cuts on beats, occasional match cuts on motion
- **Audio:** song drives everything; SFX layer per cut to glue the chaos

## Anti-pattern: mode drift

The most common editing mistake is drifting between modes without noticing. An edit that started Convergent becomes Parallel in the middle and Relay at the end — the viewer senses incoherence but can't name it.

Pick a mode. Commit. If you're going to switch modes (rare but possible), do it on an act boundary and make the switch feel intentional.

## Combining modes (advanced)

You can run a primary mode with occasional inserts from another:

- **Relay + Convergent finale** — three acts per fandom, but the last 20% intercuts all three
- **Parallel + one Convergent section** — four sweeps, then one section where they all appear together

These are advanced. Get good at single-mode edits first.

## The mode test

After drafting your structure, ask:
- Can I articulate the mode in one sentence?
- Does every section serve that mode?
- Would a viewer who's never seen any of these fandoms still feel the narrative?

If no to any of those, restructure before you start cutting.
