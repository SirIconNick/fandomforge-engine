# High-Risk Songs (YouTube copyright watchlist)

These artists and titles have a history of aggressive takedowns or monetization
claims on YouTube fan edits. When `platform_target = youtube` and the song
in `edit-plan.json` matches anything on this list, `qa.copyright` blocks
export unless the user passes an override with a documented fair-use reason.

Format is `- Artist — Title`. Both fields are matched case-insensitively as
substrings so partial matches still trigger.

This is not legal advice. Every takedown decision is case-specific. When in
doubt, test with an unlisted upload first and be ready to pull the video.

## Notes

- Instrumental-only or demetallicized versions are still claimed.
- Re-pitching by a few semitones does not dodge Content ID.
- The safer move is to license a track from Artlist / Musicbed / Epidemic /
  Soundstripe, or to use a Creative Commons track.
- If the song is absolutely required, plan for: unlisted upload, shorter than
  60s, credits block in description with transformative-use statement.
